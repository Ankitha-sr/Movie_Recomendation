Movie Recommendation System

Tools: Python · pandas · scikit-learn · NLP · Cosine Similarity

Dataset: TMDB 5000 Movies — Kaggle(https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)

Movies: 4,803 titles · Content-Based Filtering · TF-IDF Vectorization


Project Overview

Built a content-based filtering movie recommender using TF-IDF on movie genres,
keywords, cast, and overview data from a 5,000-title dataset. Compared cosine
similarity vs collaborative filtering approaches and documented trade-offs in
choosing the right method for the data available.
